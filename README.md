---------Các chức năng chính của BookStore---------

test người dùng : 
tài khoản : 012345678
mật khẩu : abc@123

test quản trị viên :
tài khoản : 087654321
mật khẩu : abc@123



* Chức năng của Người dùng
-• Đăng ký tài khoản
-• Đăng nhập
-• Chỉnh sửa thông tin tài khoản
-• Nhận thông báo khi có sách mới
-• Xem danh mục sách
-• Tìm kiếm sách
-• Xem thông tin chung về sách
-• Thêm, xóa, cập nhật số lượng sách trong giỏ hàng
-• Đặt hàng
